Fake News Detection

This project aims to detect whether a given news article is real or fake using various machine learning models. The models are trained on a dataset where each article is labeled as either real or fake based on its content and features extracted from the text. The machine learning algorithms used include Logistic Regression, Decision Trees, Random Forests, Naive Bayes, and Support Vector Machines (SVM). Users can input a news article, and the system will predict whether it is likely to be real or fake based on the trained models' predictions.
